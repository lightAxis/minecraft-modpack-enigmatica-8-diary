# 건축물

서버에 지어진 건축물을 소개하는 문서입니다.

- 모든 보기 좋게 지어놓은 건축물
- 모든 시설을 감싸기 위해 지어놓은 건축물

데코의 목적 없이 단순히 비를 피하려고 지었거나
너무 허접한 건축물은 기재되지 않습니다.
