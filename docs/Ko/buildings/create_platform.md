# Create 플랫폼

Create 시스템을 위한 모듈러 동력 전달 플랫폼.

지하에 수력 발전기 & 스팀 터빈과 메인 동력축을 연결해서, 모듈러 식으로 다양한 Create 시스템들을 구현할 수 있도록 만든 플랫폼이다.
